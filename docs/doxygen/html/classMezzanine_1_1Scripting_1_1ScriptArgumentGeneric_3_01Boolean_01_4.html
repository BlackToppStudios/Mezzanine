Detailed Description

template<>
class Mezzanine::Scripting::ScriptArgumentGeneric< Boolean >

A Bool implementation of a ScriptArgument that is suitable for primitive types in most situations.

Scripting Languages that require algorithms specific to boolean values should add required functionality to a type derived from this.
