82  /// objects inside the World. They serve as a binder between the physics and
83  /// graphics for objects and have functions that allow the manipulation of objects
84  /// loaded into the World. Currently there are 5 actor classes: ActorBase,
85  /// ActorRigid, ActorSoft, ActorTerrain, ActorCharacter. @n
