79  /// @class EventUserInput
80  /// @headerfile eventuserinput.h
81  /// @brief This is a container for MetaCodes that is used in the EventManager
82  /// @details The EventUserInput is the container for information about how a
83  /// user enters data and commands into a program. By Default one is inserted into
84  /// event manager the with all the user input from each run of the main loop.
85  /// These can be manually inserted into the EventManager to simulate input from
86  /// other sources. If setup properly this can allow computer controlled
87  /// characters to use the same interface players, allowing for more realistic
88  /// response from them. \n \n
89  /// This is implemented by inheriting from std::vector. Any feature of std::vector
90  /// can be used to interact with the metacodes stored within. Using Iterators is
91  /// great way to quickly and easily get what you need from this. For the most part
92  /// any extra functions on this are for seamless interaction with the EventManager,
93  /// or to convert incoming data into a more usable form.
94  ///////////////////////////////////////////////////////////////////////////////
