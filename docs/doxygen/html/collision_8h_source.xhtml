62  /// @headerfile collision.h
63  /// @brief This is an event class used to track collsions in the physics world.
64  /// @details This class will be used for tracking collisions in the physics world and will keep track of basic data related to the collision.
65  /// This class stores the information in the form of contact points. Often when a collision occurs there will be more then one place where
66  /// the collision occured, this is a contact point. Internally collisions only store up to a maximum of 4 contact points. When querying for
67  /// collision information, you have to provide the desired contact point index, and it must be valid. If the requested index isn't valid an
68  /// exception will be thrown. So always make sure to verify with GetNumContactPoints().
69  ///////////////////////////////////////
