Detailed Description

template<>
class Mezzanine::Scripting::ScriptArgumentGeneric< String >

A String implementation of a ScriptArgument that is suitable for primitive types in most situations.

Scripting Languages that require algorithms specific to strings should add required functionality to a type derived from this.
