321  /// @class FieldOfForce
322  /// @headerfile areaeffect.h
323  /// @brief This is field that applies force in a direction, and doesn't tamper with gravity.
324  /// @details This class is similiar to a gravity well in that it can attenuate, but different in
325  /// that the direction is constant, the source of force(for calculating attenuation) can be outside the
326  /// field itself, and the direction is constant. @n
327  /// Placing the source of attenuation inside the field will cause the object to accelerate as it gets close
328  /// to the source, and then will be applied less force(but in the same direction) as it moves from the source.
329  /// This behavior makes this class good for creating a booster-like AE.
330  ///////////////////////////////////////
